Demo AngularJS
================

Some demos for something with Angular.JS and c# .Net API.

Instalar NodeJS

Instalar o gerenciador de pacotes Bower

Instalar os pacotes:
* Jquery - 
* AngularJS - https://angularjs.org/

* MomentJS - http://momentjs.com/
